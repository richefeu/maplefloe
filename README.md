# What is `maplefloe`?

`maplefloe` is a C++ application for simulating the dynamics of floating ice floes. It provides tools for modeling interactions between floes, including collision detection, contact forces, and packing algorithms.

> It is actually being developped for now. So it is not yet ready!


## Features

- **Floe Dynamics**: Simulate the motion and interaction of individual ice floes
- **Collision Detection**: Efficient algorithms for detecting collisions between floes
- **Contact Forces**: Calculate forces between colliding floes
- **Packing Algorithms**: Optimize the arrangement of floes in 2D and 3D space
- **Geometric Utilities**: Tools for working with 2D and 3D geometry
